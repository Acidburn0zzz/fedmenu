Try clicking the button at the bottom of the page (the Fedora logo)

This is a little demo of a menu that we can add to all our apps.

The javascript code to make this menu happen looks like this..

<script src="http://code.jquery.com/jquery-1.11.2.min.js"></script>
<script src="fedora-libravatar.js"> </script> <!-- only needed for avatars -->
<script src="fedmenu.js"> </script>
<script>
 fedmenu({
   /* Optional. Default is 'bottom-left' */
   'position': 'bottom-right',

   /* Optional. Use this if the page is "about" a user.. like on
     their badges page. Do not use it for a logged in user. */
   'user': 'kevin',

   /* Optional. Use this if the page is "about" a package.. like
     on the pkgdb page for a package. */
   'package': 'firefox',
 });
</script>
